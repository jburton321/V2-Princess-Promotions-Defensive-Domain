## What Makes This Competition-Grade

- Word-by-word hero animation with staggered delays
- Score counters that animate from 0 to target on scroll
- Scroll progress bar tracking page position
- Header transitions from transparent to frosted glass
- Issue bento with 6 distinct card types (navy hero, image bg, stat, quote, standard, image wide)
- FCC journey bento with 5 card types (phone hero, stat, action, warning, image)
- Horizontal scroll testimonial gallery
- CSS grid accordion (not max-height hack)
- Full-width typographic "Is It a Scam?" moment
- Spring-easing scroll reveals with staggered cascades
- All headings in Title Case
- One unified 1100px content width
- No bullet points or numbered lists anywhere
- 3 JSON-LD schema blocks for SEO rich snippets
- 9 Open Graph tags + 4 Twitter Card tags
